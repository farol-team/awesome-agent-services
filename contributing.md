# Contributing

Your contributions are always welcome!

## Guidelines

- **Agent-first only**: The service must be designed primarily for AI agents, not adapted from a human-first product. If a human is the primary user, it doesn't belong here.
- **Working product**: Must have a working product or public API. No vaporware, no "coming soon".
- **Add at the bottom** of the appropriate category.
- **Format**: `| [Name](URL) | One-line description — what it does for agents | Funding / Status |`
  - Description starts with uppercase, no period at end (table format)
  - Be specific about what makes it agent-native
- **No duplicates**: Search existing entries before submitting.
- **No archived/unmaintained projects**: If the repo is archived or the service is shut down, don't add it.

## How to add an entry

1. Fork this repository
2. Add your entry to the appropriate category (alphabetical order within category)
3. Submit a pull request with a brief explanation of why it qualifies as agent-native infrastructure

## Criteria checklist

- [ ] Primary user is an AI agent, not a human
- [ ] Has a public API or SDK
- [ ] Active and maintained
- [ ] Not already listed
